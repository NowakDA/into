2
Открытие инспектора элементов

Найдите элемент, который хотите изменить (например, столбец в таблице):

  1. Наведите курсор на нужный элемент таблицы
  2. Нажмите правой кнопкой мыши
  3. В контекстном меню выберите "Посмотреть код" (или "Inspect" в английской версии)

Пример таблицы "Профиль абонента: Таблица по тарифам"

3
Поиск нужных классов и ID

В инспекторе кода вы увидите HTML-структуру. Ищите следующие элементы:

Что искать в коде:

  • div.chart-container.css-2yffut - контейнер графика
  • dashboard-chart-id-150 - уникальный ID вашего чарта
  • cell-bar positive css-1fjbudt - класс ячейки с данными
💡 Обратите внимание на класс cell-bar - именно его мы будем стилизовать для изменения цвета столбцов.

5
Применение CSS стилей

Вставьте следующий код в CSS редактор для изменения цвета столбцов:

/* Изменение цвета столбцов в таблице */
.dashboard-chart-id-150 .cell-bar {
  background-color: red !important;
}

/* Альтернативный вариант с градиентом */
.dashboard-chart-id-150 .cell-bar {
  background: linear-gradient(90deg, #ff4444, #ff6666) !important;
}

Дополнительные CSS стили

Примеры других полезных стилей для вашего дашборда:

/* Стили для заголовка дашборда */
.dashboard-markdown h1 {
  font-size: 30px !important;
  color: #ff3124 !important;
  text-align: center;
}

/* Настройка внешнего вида таблицы */
.ant-tabs-content-holder {
  color: #5c5c5c !important;
  font-size: 12px;
}

/* Градиентный фон для компонента */
.dashboard-component {
  background-image: linear-gradient(-45deg, #2D90D4, #D290D6);
}
